# 🧠 AI Resume Builder

An AI-powered web app that generates professional resumes and cover letters using Streamlit.

## 🚀 Features
- Easy form-based input
- Auto resume generation
- Cover letter creation
- Clean UI with Streamlit

## 🛠️ Tech Stack
- Python
- Streamlit

## 📦 Installation

```bash
pip install -r requirements.txt
streamlit run app.py
